# Configuration

create myWifi.h in src include with the following content, containing your wifi credentials:

```cpp
#ifndef MYWIFI_H_
#define MYWIFI_H_

#define MY_WIFI_SSID "<ssid>"
#define MY_WIFI_PASSWORD "<wifi_password>"

#define NTP_SERVER "pool.ntp.org"

#endif
```
